DoodleJump

I created a version of the classic game "Doodle Jump" using MIPS Assembly. Players control a character that must jump from platform to platform to avoid falling into the abyss. The game features power-ups, dynamic notifications, and original music.

This was submitted for the final assessment of CSC258 (a computer hardware course at UofT) and received a perfect score.

ObjectivelyBestCalendar

I worked on a calendar application built with Java and Java Swing. The design process focused on following the SOLID principles of object-oriented programming. Users can create and customize events, series, notifications and alerts. They can create multiple calendars on multiple accounts.

I worked in a group of five for this project and we submitted the application for the final assessment of CSC207 (a software design course at UofT).
